**About the Game**

Firewall Defenders is a 3D third-person shooter built in Unity 6 using C#. The player controls a security agent inside a sci-fi network corridor, defending the system's Firewall from incoming waves of
malicious software agents represented as digital spiders. The game is built around the course theme of introducing computer science concepts through gameplay: before each wave, an educational panel 
explains a real network threat such as a virus, worm, DDoS attack, Trojan horse, or ransomware, paired with a clear objective for the player.
The game includes a main menu with an animated glitch intro sequence, a fully playable game scene with increasing wave difficulty, a dynamic firewall health bar with visual feedback, background music 
with scene-transition fade, and an animated game over screen. The project was developed collaboratively by four members, with each taking primary responsibility for one significant programming challenge.

**Computer Science Concept**

The game introduces the following cybersecurity concepts through its wave system:
•	Viruses
•	Worms
•	DDoS Attacks
•	Trojan Horses
•	Ransomware
•	Firewalls
Each concept is presented through an in-game panel with a description and a gameplay objective, embedding education directly into the game loop.
